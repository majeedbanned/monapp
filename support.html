Support

This page is the public support surface for the current Monely website build. It covers the most common product questions and launch-time troubleshooting topics. Last updated on March 29, 2026.

Getting started

Monely is built around a straightforward order of operations: capture a receipt, confirm the structured fields, review the month, then keep assets or shared-ledger activity in the same workspace.

  • Grant camera access if you want to scan paper receipts.
  • Use photo-library access for saved images or screenshots.
  • Import PDFs directly into the same receipt workflow.
  • Set your preferred default currency early so reports and assets speak one language.

Receipt capture help

If a receipt does not parse the way you expect, keep the original image or PDF attached, confirm the merchant, date, and total manually, and only run AI analysis when it actually improves the entry.

  • Use brighter images and flatter receipts for cleaner OCR.
  • Imported PDFs may still need OCR when text extraction is limited.
  • Manual confirmation remains part of the intended workflow.

Reports and monthly insight

The app surfaces multiple focused views instead of one giant dashboard, so support questions often come down to choosing the right report for the question you are asking.

  • Use monthly reports for totals and category mix.
  • Use compare-months views to understand what changed.
  • Use forecast views for recurring load and future movement.

Shared ledger help

Shared finance workflows are designed for households and small teams. If a ledger feels read-only, verify that collaboration is active, the correct ledger is selected, and the current user has edit permissions.

  • Check whether the current session is read-only.
  • Confirm the active ledger and the current member role.
  • Review invite or redemption flows if another member cannot join.

Assets and market data

Asset views can fetch pricing from configured providers. If freshness looks stale, refresh market data and confirm the selected symbol, provider support, and currency settings.

  • Alpha Vantage is the primary provider for supported stocks and commodities.
  • CoinGecko can be used as a fallback for selected crypto assets.
  • Price freshness indicators matter and should not be ignored.

Backup and restore

Backups exist so the finance workflow does not depend on blind trust. Before large changes, create a backup, verify the file exists, and only then clean up older data if needed.

  • Create backups before testing restore flows.
  • Keep at least one known-good export before major device changes.
  • After a restore, review totals and the active ledger context.

Privacy and data handling

Monely is local-first, but not fully offline for every feature. AI analysis, shared-ledger sync, and live market data can call configured services. Use the dedicated hosted policy for the full statement.
